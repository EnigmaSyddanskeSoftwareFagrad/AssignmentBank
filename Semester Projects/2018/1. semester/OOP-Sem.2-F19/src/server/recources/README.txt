Rules for making or changing config files

* Parameters are written as [Parameter] := [Value]
* The spacer should always be " := " space,colon,equals,space
	Example:
		title := theTitleStored

* Line changes always have to be made with both CR(0D) and LF(0A) *Here denoted by their name and hex value
* There should be no empty lines in the config file.
* Config files must always use the extension ".config"
